## Liste et Description des fonctionnalités IA

**Correcteur automatique d'orthographe**: cette fonctionnalité permet de détecter et de corriger automatiquement les erreurs de frappe et les fautes d'orthographe dans les textes saisis par l'utilisateur. Elle analyse chaque mot en temps réel ou après la saisie, le compare à un dictionnaire linguistique, puis propose la correction la plus probable.

Le correcteur prend en compte les erreurs courantes telles que les lettres manquantes, inversé ou mal orthographié, et peut suggerer plusieurs alternatives lorsque plusieurs corrections sont possibles. Il améliore ainsi la qualité des textes, réduit les erreurs et facilite la saisie, notamment pour les utilisateurs non experts ou dans un contexte de rédaction rapide.
